To start the project the file Hierarchy must be like this:
|
|-> Main folder of the project
|-> JSONconf/

The JSONConf can be find in the deliverables folder, it has to be copied as described above.

The user has to first start the Server.jar that can be found in Deliverables and then start the Client.jar processes.
